# application-TIM-DATR

Site web pédagogique TIM pour étudiants de BTSA DATR 1ère et 2ème année.

## Lancer localement

Ouvrir `index.html` dans un navigateur.

## Structure

- `index.html` : accueil et positionnement référentiel.
- `cours/` : page principale + 8 sous-pages thématiques TIM.
- `exercices.html` : activités contextualisées.
- `corriges.html` : corrections justifiées.
- `projets.html` : scénarios M4/M6.
- `ccf.html` : sujets CCF, consignes, grille.
- `assets/` : styles et script.
